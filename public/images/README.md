# Bilder-Platzhalter

Ersetze diese SVG-Platzhalter mit deinen eigenen Bildern:

| Datei | Größe | Verwendung |
|-------|-------|------------|
| `hero-bg.svg` → `hero-bg.jpg` | 1920×1080 | Hero-Hintergrund |
| `portrait.svg` → `portrait.jpg` | 800×800 | Über-Mich Portrait |
| `service-1.svg` → `service-1.jpg` | 800×600 | Leistung 1 |
| `service-2.svg` → `service-2.jpg` | 800×600 | Leistung 2 |
| `service-3.svg` → `service-3.jpg` | 800×600 | Leistung 3 |
| `logo-symbol.svg` → `logo-symbol.png` | 100×100 | Logo im Header |

## Anleitung

1. Eigene Bilder vorbereiten (empfohlene Größen beachten)
2. Platzhalter-SVGs löschen
3. Eigene Bilder mit gleichen Namen hochladen
4. In `index.astro` ggf. Dateiendungen anpassen (.svg → .jpg/.png)

## Tipp

Für beste Qualität:
- Hero-Hintergrund: JPEG, komprimiert
- Portrait: JPEG oder PNG
- Service-Bilder: JPEG
- Logo: PNG mit Transparenz
